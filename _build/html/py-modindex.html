Admin:

• Login: Admin can login by using correct credentials.

• View Users: Admin can view all users.

• Verify Users: Admin can verify users.

• Add Voters Data: Admin can add new data of voters, district, local body, ward and polling station.

• View Voters: Admin can view all voters

Local Member:

• Login: Local Members can login by using correct credentials.

• View Voters: Local Members can view all voters.

• Report Issue: Local Members can report any issues.

• View Reply: Local Members can view replies to issues.

• Add Suggestions: Local Members can share their suggestions.

• View Tasks: Local Members can view assigned tasks.

Council Member:

• Login: Council Members can login by using correct credentials.

• View Voters: Council Members can view all voters.

• View Filtered Voters: Council Members can view voters filtered data with more details which local member added.

• Report Issue: Council Members can report any issues.

• View Reply: Council Members can view replies to issues.

• Add Suggestions: Council Members can share their suggestions.

• View Tasks: Council Members can view assigned tasks.

Super Member:

• Login: Super Members can login by using correct credentials.

• View Voters: Super Members can view all voters.

• View Filtered Voters: Super Members can view voters filtered data with more details which local member and council member added.

• Report Issue: Super Members can report any issues.

• View Reply: Super Members can view replies to issues.

• Add Suggestions: Super Members can share their suggestions.

• Assign Tasks: Super Members can assign tasks to local and council members.

President:

• Login: President can login by using correct credentials.

• View Voters: President can view all voters.

• View Filtered Voters: President can view voters filtered data with more details which local member and council member added.

• Report Issue: President can report any issues.

• View Reply: President can view replies to issues.

• Add Suggestions: President can share their suggestions.

• Assign Tasks: President can assign tasks to local, council, and super members.

• View Assigned Tasks: President can view all assigned tasks.

• Voting Probability: President can view voting probability.
